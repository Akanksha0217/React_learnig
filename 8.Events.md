Events are actions performed by the user, such as:
- Clicking a button
- Typing in input
- Submitting a form
- Moving mouse

React handles events using camelCase syntax.

| onKeyDown   | Key press    |


## React Events vs HTML Events
| React      | HTML        |
| ---------- | ----------- |
| onClick    | onclick     |
| camelCase  | lowercase   |
| JSX syntax | HTML syntax |
